# Online Bookstore Management System

This is a web application for managing an online bookstore. It allows users to browse books, add them to a cart, and place orders. Admin users can manage the inventory of books.

## Features
- **Admin**: Add, update, and delete books.
- **Customer**: Browse books, add to cart, and place orders.

## Technologies Used
- ASP.NET MVC
- ASP.NET WebAPI
- SQL Server
- Entity Framework Core
- Bootstrap

## How to Run
1. Clone the repository.
2. Open the solution in Visual Studio.
3. Set up the database using Entity Framework migrations.
4. Run the application.
